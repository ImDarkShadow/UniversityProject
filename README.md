# Automatic Comic Translator
##### Prerequisites:
- Tesseract OCR
- Python <= 3.7
- pip
- NodeJS <=14
- npm

### Installation
- `git clone https://github.com/ImDarkShadow/UniversityProject`
- `npm install`
- `pip install -r requirements.txt`

### Uses
- Copy comic chapter zip inside raw folder
- `npm start`
- output at output folder as zip
